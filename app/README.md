# tech-role-site (App)

概要:
- Vite + React + Tailwind の雛形です。診断のフロント実装をここに進めます。

セットアップ:
1. Node.js をインストール (推奨 v18+)
2. ターミナルで `cd app` を実行
3. `npm install` を実行
4. `npm run dev` でローカル起動

次のタスク:
- 診断フローの各コンポーネント（入力フォーム、質問スライダー、結果表示）を実装
- Framer Motion と Lottie を導入してアニメーションを追加
- 環境に応じて Supabase/Firebase の接続を検討（結果保存・管理画面）
